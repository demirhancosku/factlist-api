## About

The Factlist API service is built with Nodejs, Graphql and latest stable versions of other npm packages. 

## Getting started

### Development

- Install npm modules:
  ```
  yarn install
  ```

- Create database migration
  ```
  yarn db:migration
  ```
  
- Create database seed
  ```
  yarn db:seeders
  ```

- Start server watch 
  ```
  yarn watch
  ```
